User Guide

1. Start Recognition: Clicking this allows starts the speech Recognition and consequently the transcription. It is the blue button.

2. Stop Recognition: Clicking this allows you to stop the speech Recognition and the transcription process. It is the red button.

3. Save Output: Clicking this allows you to save the recognized speech transcription. It is the green button.

4. Manage Dictations: This takes you to a page to properly format or edit saved transcriptions and also to be able to delete them. It is the grey button.
